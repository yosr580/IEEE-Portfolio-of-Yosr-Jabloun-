# Yosr Jabloun - IEEE Portfolio

Welcome to the personal portfolio of **Yosr Jabloun**, a Telecom Engineering Student, IEEE Changemaker, and the 2025 Rising Star of the IEEE Tunisia Section.

## 🌟 About This Project

This project is a modern, responsive, and visually striking web portfolio designed to showcase Yosr's journey, leadership, and impact within the IEEE community. It features a dark-themed, futuristic "glassmorphism" aesthetic with vibrant neon highlights.

## ✨ Features

- **Modern UI/UX Request:** Responsive grid layouts, glass-card elements, and neon-glow details.
- **Bento Gallery:** A beautifully styled "Bento box" image gallery to display favorite IEEE moments and events.
- **Dynamic Content Sections:** Highlighting Profile, IEEE Leadership, Major Events, Educational Activities, and Awards.
- **Performance Optimized:** Built with pure HTML and CSS, utilizing flexbox and CSS Grid architecture.

## 🛠️ Technologies Used

- **HTML5:** Semantic structure.
- **CSS3:** Custom styles, CSS variables, flexbox, grid, animations, and glassmorphism.
- **Google Fonts:** Montserrat and Inter for sleek, readable typography.

## 🚀 How to Run Locally

1. Clone this repository to your local machine.
2. Open the project folder.
3. Double-click the `index.html` file to open it in your default web browser. No server setup or build step is required!
